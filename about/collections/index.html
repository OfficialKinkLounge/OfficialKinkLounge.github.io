Terminology/Criteria


For a collection to be officially proclaimed, it must...

-Have a minimum of three base members, all of which are in the Kink Lounge.
-All members must have their given christian names starting with the same character.
-And the three main positions must be filled: Big, Medium and Lil.

If these criteria are met, the tallest base member will become "Big [character]" the shortest will be "Lil [character]" and the middle member will be proclaimed "Medium [character]".
